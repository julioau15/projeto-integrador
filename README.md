# Projeto Integrador
Projeto desenvolvido no SENAI integrando as disciplinas de Arquitetura de Redes, Sistemas Operacionais, Lógica de Programação, Levantamento de Requisitos e Linguagem de Marcação. O objetivo é aplicar, de forma prática, os conceitos fundamentais de infraestrutura, software e desenvolvimento, passando pela análise de requisitos, modelagem da solução, lógica de funcionamento, estruturação do sistema e sua apresentação em interface web, simulando um projeto real do ambiente profissional de TI.
